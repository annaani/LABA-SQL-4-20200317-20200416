Домашнее задание к Занятию 3
Функции агрегации

Примечание: всем аггрегатам дать псевдонимы

1. Выведите минимальное и максимальное значение для заказов в разрезе
   ID заказа (таблица order_items колонки unit_price * quantity) 
   для товаров продаваемых более 100 единиц (подсказка WHERE) (2 балла)

2. Выведите общую сумму и количество позиций для заказов в разрезе 
   ID заказа (таблица order_items колонки unit_price * quantity)
   для товаров продаваемых в среднем более 100 единиц (подсказка HAVING) (3 балла)

3. Выведите максимальное и минимальное значение цены для каждой категории (таблица products) 
   для товаров цена на сайте которых в промежутке 200 - 750 (3 балла)

4  Выведите количество сотрудников и количество должностей в компании (таблица employees) (3 балла)

5. Выведите суммы продаж в разрезе ID продукта (таблица order_items) 
   для продуктов со средней продажей не менее 140шт
   Отсортируйте по убыванию суммы продаж (3 балла)

6. Выведите ID заказа и количество позиций в чеке (таблица order_items) 
   для заказов на сумму свыше 1 млн и средним значением не выше 100 тыс 
   Отсортируйте по номеру заказа (3 балла)

7. Выведите количество заказов, идентификатор продавца и год продажи используя функцию YEAR, (таблица orders)
   для заказов в статусе 'Отправлено' или 'В ожидании' (примечание: 'Shipped', 'Pending')
   Отсортируйте по году возрастанию, затем количество заказов по убыванию, затем ID продавца по возрастанию. (3 балла)
